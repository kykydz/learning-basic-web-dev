Cara 1: Upload via Website GitHub (Manual Upload)
Cocok untuk file kecil atau hanya satu dua file.

Buka https://github.com dan login.

Buat repositori baru dengan klik tombol “New” di halaman repo.

Beri nama repositori dan klik "Create repository".

Setelah repo dibuat, klik tombol “Add file” → “Upload files”.

Klik “Choose your files” dan pilih file dari komputer kamu.

Tambahkan pesan commit (misalnya: Initial upload).

Klik tombol “Commit changes”.

Cocok untuk project lengkap dari komputer lokal.

📌 Persyaratan Awal:
Sudah install Git

Sudah punya akun GitHub dan membuat repo kosong

# 1. Buka terminal / Git Bash
cd path/ke/folder/projekmu

# 2. Inisialisasi git di folder
git init

# 3. Tambah semua file ke staging
git add .

# 4. Commit perubahan
git commit -m "Initial commit"

# 5. Hubungkan ke repo GitHub (ganti URL dengan repo milikmu)
git remote add origin https://github.com/username/nama-repo.git

# 6. Upload ke GitHub
git push -u origin master
